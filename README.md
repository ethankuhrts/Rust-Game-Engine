# Rust-Game-Engine
A game engine built with the Rust programming language.

NOT ANYWHERE NEAR A PRODUCTION STATE

Dependencies
 - WGPU for graphics 
 - Winit for windowing and input
 - GLAM for math
 - Specs ECS

Plans 
 - Lighting
 - Implement Rapier physics
 - Asset Loading
 - UI
 - Particles
 - Android support
 - Scene Editor
